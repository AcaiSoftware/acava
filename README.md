# Acava: Java Library for Acai

## Add Acava to your own build
### Maven
```xml
<repository>
  <id>acava</id>
  <url>https://raw.github.com/AcaiSoftware/acava/repository/</url>
</repository>

<dependency>
  <groupId>gg.acai</groupId>
  <artifactId>acava</artifactId>
  <version>1.0.1</version>
</dependency>
```
